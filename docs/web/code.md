# Web Code

Write here your own content!

## Front-end

The front-end of the website is written in HTML, CSS and Javascript

---

The front-end gets it's data from the back-end by using Javascript HTTP calls.

---

All externals (JS, CSS, images) are stored in its own files and included in the HTML.

---

## Back-end

The code of the back-end is written in PHP, following the PSR-12 coding style.

--- 

There is usage of include, functions and classes to prevent repetition of code.

---

There is an API which is used by the front-end and embedded device. The API is the only entrance to the database.

---

The back-end returns JSON-formatted text, always. So even if you add new data, you get a response in JSON.

---
